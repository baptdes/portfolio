# My portfolio

It's just my beautiful portfolio, built with SvelteKit and Tailwind CSS.

## Contributing

It's a portfolio, why would you want to contribute? But if you do, open a PR and I'll take a look by curiosity.

## Development

Install dependencies with `pnpm i`, start a development server:

```sh
pnpm run dev

# or start the server and open the app in a new browser tab
pnpm run dev --open
```

## Building

To create a production version of your app:

```sh
pnpm run build
```

You can preview the production build with `pnpm run preview`.
